Study Tracker

Site Name

This name (Study Tracker) represents a tool designed to help students organize and track their study tasks.

Site Purpose

This website helps users manage their study tasks, track progress, and improve productivity through a simple interface.

Scenarios

  • How can I add and manage my study tasks?
  • How can I track which tasks I have completed?

Color Schema

Primary color (#6204BF) is used for headings and buttons.

Secondary color (#0339A6) is used for accents.

Background color (#F2F2F2) is used for the page background.

Typography

Primary font: Montserrat (used for headings)

Secondary font: Quicksand (used for body text)
